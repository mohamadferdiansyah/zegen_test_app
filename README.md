# Zegen Store - Modern E-commerce App

Zegen Store adalah aplikasi e-commerce modern yang dibangun menggunakan **Flutter** dan **GetX**. Aplikasi ini menggunakan **Fake Store API** untuk data produk dan mensimulasikan alur belanja lengkap dengan desain yang bersih dan responsif.
